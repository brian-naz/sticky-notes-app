# Sticky Notes App

To get started:
<a href="https://sticky-notes-app-seven.vercel.app/">Demo</link>

### Setup:

Clone the repo:
```bash
git clone https://github.com/brian-naz/sticky-notes-app.git
```
Go to the directory and simply type:
```bash
npm install
```
To run:
```bash
npm start
```
